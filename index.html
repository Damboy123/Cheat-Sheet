HTML ANATOMY

HTML ELEMENT Description
< p > this is an open html tag
</ p > this is a close html tag
<p>This is a paragraph</p> together, they are called an html element

HTML TAGS

Opening Tag Content Closing Tag Description
<p> words </p> this provides a paragraph in a webpage
<html> <body> ... </body> </html> this is the root element of an HTML document
<head> links like css, <title>tag </head> this contains the metadata of the webpage
<title> Title of Webpage </title> title in the browser title bar
<body> <table> .... </table> </body> contains all the contents of HTML document
<table> thead, th, tbody,tr,td </table> this makes a table in html
<thead> <th>... </th> </thead> use to group the header of the table
<th> Table header </th> this provides a table header data
<tbody> <tr>... </tr> </tbody> use to group the body of the table
<tr> <td>... </td> </tr> use to provide a row of the table
<td> </td> use to give data to the row of the table
<a href="link to another webpage or another area of the site"> This is a link! </a> provides link to other webpage
<video src = "video url or address" width ="320" height = "240" controls> Video not Supported </video> use to display video
<img src="image-location.png" alt = "This is an image"/> display images
